# Welcome to Nalanda

Nalanda is a **knowledge engine** designed for building living blogs where ideas connect and grow into a graph of understanding.

## What Makes Nalanda Different?

Traditional blogs present content in reverse chronological order. Nalanda organizes knowledge as an interconnected graph, making it easier to:

- Discover related concepts
- Follow idea trajectories
- Build a personal knowledge base

## Key Features

### 1. Backlinks

Use the `[[concept name]]` syntax to create automatic connections between posts. For example, when you write about [[Machine Learning]], Nalanda automatically:

- Creates a concept page for Machine Learning
- Links this post to that concept
- Shows this post as a backlink on the concept page

### 2. Knowledge Graph

Visit the `/graph` page to visualize how your content connects. The graph shows:

- **Posts** (blue nodes)
- **Concepts** (green nodes)  
- **Tags** (purple nodes)

### 3. Content Types

Nalanda supports multiple content types:

- **Posts**: Standard blog posts
- **Ideas**: Quick thoughts and notes
- **Research**: In-depth research notes

## Getting Started

1. Create your first post using the visual editor
2. Use `[[concept]]` syntax to link ideas
3. Explore the graph to see connections emerge
